# HU-001: Reconocimiento de Patentes mediante OCR

**Como:** Usuario del sistema  
**Quiero:** Que el sistema reconozca los caracteres de las patentes de vehículos en imágenes  
**Para:** Obtener el texto plano de la matrícula y poder procesarlo automáticamente  

## Criterios de Aceptación

| Escenario | Dado que (Given) | Cuando (When) | Entonces (Then) |
|-----------|------------------|---------------|-----------------|
| **Reconocimiento Exitoso** | El sistema recibe una imagen clara con una patente visible | El módulo de OCR procesa la imagen | El sistema devuelve el texto plano de la patente correctamente (ej: "AB123CD") |
| **Sin Patente Detectada** | Se proporciona una imagen que no contiene ninguna patente | El módulo de OCR intenta procesarla | El sistema retorna un mensaje indicando que no se detectó ninguna patente |
| **Imagen de Baja Calidad** | Se proporciona una imagen borrosa, oscura o con la patente obstruida | El módulo de OCR intenta leer los caracteres | El sistema retorna un error de lectura o un nivel de confianza bajo, solicitando una nueva imagen |
| **Extracción de Texto** | Se ha detectado una patente válida | El proceso de reconocimiento finaliza | El resultado se disponibiliza exclusivamente en formato de texto plano sin caracteres especiales extraños |
